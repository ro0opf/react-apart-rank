### 기능
* 검색(자동완성)
* 검색어 순위
* 티어정보 (지역별, 전체)
* 지역별 순위
* 전체 순위
* SNS 공유
* 검색한 지역 대장 아파트
* 전국 대장 
* 검색 결과
   * 우리동네 (대장 & 저렴) / 전국 (대장 & 저렴)
* 최근 거래일 기준 가장 많이 오른 아파트
